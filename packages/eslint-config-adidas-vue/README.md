# eslint-config-adidas-vue

Adds ESLint settings and rules for Vue framework.

## Install

```
npm i --save-dev eslint@9 eslint-plugin-vue@9 eslint-config-adidas-vue
```

## Usage

```json
{
  "extends": [
    "adidas-vue"
  ]
}
```

Usually you would mix this with other eslint configurations, like:

```json
{
  "extends": [
    "adidas-env/browser",
    "adidas-env/module",
    "adidas-es6",
    "adidas-vue"
  ]
}
```
